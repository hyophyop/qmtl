# 시작하기

!!! abstract "QMTL이란?"
    QMTL은 트레이딩 전략을 작성하고 제출하면, 시스템이 **자동 백테스트·정책 평가·승격/강등 결정을 목표로** 설계된 플랫폼입니다.  
    현재는 자동 백테스트와 정책 기반 평가/강등이 구현되어 있으며, **자동 승격/자본 배분/실시간 모니터링**은 단계적으로 도입 중입니다.

## 핵심 가치

> **"전략 로직에만 집중하면 시스템이 알아서 최적화하고 수익을 낸다."**

```
┌─────────────────────────────────────────────────────────────┐
│                    사용자가 하는 것                          │
│                                                             │
│   1. 전략 작성      2. 제출 (한 줄)    3. 결과 확인          │
│   ┌─────────┐      ┌─────────┐       ┌─────────┐          │
│   │ class   │ ──▶  │ submit()│ ──▶   │ 성과표  │          │
│   │ MyStrat │      └─────────┘       │ 순위    │          │
│   └─────────┘                        │ 배분액  │          │
│                                      └─────────┘          │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                  시스템이 맡는 일                             │
│   (사용자가 내부를 알 필요 없음)                               │
│                                                             │
│   • 백테스트 실행 및 히스토리 리플레이 (구현됨)               │
│   • 성과 평가 (Sharpe, MDD, 상관관계 등) (구현됨)            │
│   • 유효성 판단 및 강등 처리 (구현됨)                         │
│   • 승격/자본 배분/실시간 모니터링 자동화 (순차 도입 예정)    │
└─────────────────────────────────────────────────────────────┘
```
