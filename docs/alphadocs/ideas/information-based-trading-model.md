📌 시장 미시 구조 이론: 정보 기반 거래 모델 (Information-Based Trading Model)

🏛 이론 개요

정보 기반 거래 모델(Information-Based Trading Model)은 시장 참가자들이 비대칭적 정보를 보유하고 있으며, 이들이 매매하는 과정에서 가격이 점진적으로 조정된다는 개념에 기반합니다.
	•	특정 투자자(예: 기관 투자자, 내부자 거래자)는 사전 정보를 보유하고 있으며, 이들이 거래할 때 시장 유동성이 비대칭적으로 변함.
	•	시장 조성자(Market Maker)는 주문 흐름을 분석하며 가격을 업데이트하지만, 정보 보유자의 행동을 완전히 예측할 수 없음.
	•	결과적으로 시장 가격은 비선형적인 방식으로 정보가 반영되면서 이동하게 됨.

📌 대표적인 모델:
	1.	Kyle Model (1985)
	•	정보 기반 투자자가 유동성 투자자와 시장 조성자 사이에서 거래하며, 시장 조성자는 주문 흐름을 분석해 가격을 조정함.
	•	시장 충격 함수는 비선형적이며, 정보가 점진적으로 반영될수록 가격 변화 패턴이 달라짐.
P_t = P_{t-1} + \lambda Q_t + \epsilon_t
	•	\lambda는 유동성 계수, Q_t는 거래량, \epsilon_t는 노이즈.
	2.	Glosten-Milgrom Model (1985)
	•	매수·매도 주문이 들어올 때마다 시장 조성자는 정보 기반 거래 여부를 확률적으로 추정하여 호가를 조정.
	•	정보 기반 주문이 많아질수록 스프레드가 확대됨.

📈 퀀트 알파 응용

✅ 비선형 정보 흐름 기반 전략
	1.	주문 흐름 불균형(Order Flow Imbalance, OFI) 활용
	•	특정 거래소에서 매수 주문이 지속되거나 매도 주문이 몰리는 경우, 정보 기반 투자자의 존재를 추정.
	•	비선형 함수 적용:
OFI_t = \frac{\sum_{i=1}^{N} \max(0, Q_i)}{\sum_{i=1}^{N} |Q_i|}
	•	OFI_t가 특정 임계값 이상이면 롱(매수), 이하이면 숏(매도) 진입.
	2.	스프레드 변화 기반 변동성 예측
	•	정보성 거래자가 많아질수록 호가 스프레드가 커지므로, 변동성을 예측하는 데 활용 가능.
	•	비선형 모델 적용:
\sigma_t^2 = w_1 \cdot \text{Spread}_t^{\alpha} + w_2 \cdot \text{Liquidity}_t^{\beta}
	•	\alpha > 1이면 정보 기반 거래자가 증가할 때 변동성이 급격히 상승.
	3.	시장 충격 기반 스캘핑 전략
	•	특정 수준 이상 가격이 상승(또는 하락)한 후, 정보 기반 트레이더들이 포지션을 정리할 때 평균 회귀(mean reversion) 발생 가능.
	•	비선형 가격 반응 모델:
\Delta P_t = \lambda Q_t^\gamma
	•	0 < \gamma < 1이면 가격 조정이 완만하게 일어남 → 스캘핑 전략 가능.

⸻

🔍 추가 고려 사항
	•	틱 데이터(Level II) 활용 필수 → 비대칭 정보 감지를 위한 고해상도 분석 필요.
	•	시장 조성자와의 경쟁 고려 → 유동성 공급자가 빠르게 대응하는 시장에서는 신호 왜곡 가능.
	•	옵션 시장과의 연계 분석 → 옵션 미결제약정(Open Interest) 및 변동성 스마일 분석으로 신호 신뢰도 증가.

📢 결론
정보 기반 거래 모델을 활용한 비선형 퀀트 전략은 주문 흐름 분석, 변동성 예측, 시장 충격 모델링에 강력한 도구가 될 수 있으며, 특히 고빈도 트레이딩(HFT) 및 유동성 기반 전략에서 활용도가 높다. 🚀
