📌 시장 미시 구조 이론: 비대칭 정보 이론 (Asymmetric Information Theory)

🏛 이론 개요

비대칭 정보 이론(Asymmetric Information Theory)은 시장 참여자들이 동일한 정보를 갖고 있지 않다는 가정에서 출발합니다.
일부 투자자는 시장의 주요 정보(예: 기업 실적, 거시경제 지표, 중앙은행 정책 변화)를 미리 알고 있으며, 이를 이용해 거래를 수행할 수 있습니다.
이러한 정보 비대칭성은 유동성 공급자와 정보 기반 트레이더 사이의 불균형을 초래하여 시장 가격 움직임에 영향을 미칩니다.

📌 대표적인 모델:
	1.	Kyle Model (1985)
	•	정보 보유자가 유동성 트레이더와 시장 조성자 사이에서 거래하며, 시장 조성자는 가격을 업데이트하며 정보의 존재를 점진적으로 반영함.
	•	시장 충격 함수는 비선형적이며, 정보가 점진적으로 시장에 반영될수록 가격 변화의 패턴이 달라짐.
P_t = P_{t-1} + \lambda Q_t + \epsilon_t
	•	여기서 \lambda는 시장 유동성을 나타내며, Q_t는 거래량, \epsilon_t는 노이즈.
	2.	Glosten-Milgrom Model (1985)
	•	매수·매도 주문이 들어올 때마다 시장 조성자가 확률적으로 정보성 거래 여부를 판단하며, 이에 따라 호가 스프레드를 조정.
	•	정보성 주문이 많을수록 스프레드가 확대되며, 시장이 비효율적으로 반응할 수 있음.

📈 퀀트 알파 응용

✅ 비선형 정보 기반 전략
	1.	비대칭적 주문 흐름 분석
	•	특정 거래소에서 이상적으로 높은 매수 주문이 지속되거나, 매도 주문이 몰리는 경우, 정보 기반 투자자의 존재를 추정할 수 있음.
	•	비선형 함수 적용:
\text{OFI}t = \frac{\sum{i=1}^{N} \max(0, Q_i)}{\sum_{i=1}^{N} |Q_i|}
	•	OFI_t: Order Flow Imbalance (주문 흐름 불균형)
	•	Q_i: i번째 거래량 (매수는 양수, 매도는 음수)
	•	OFI_t가 특정 임계값 이상이면 롱(매수), 이하이면 숏(매도) 진입
	2.	호가 스프레드 기반 변동성 예측
	•	정보성 거래자가 많아질수록 스프레드가 커지므로, 스프레드 변화를 활용한 옵션 변동성 예측 가능
	•	비선형 모델:
\sigma_t^2 = w_1 \cdot \text{Spread}_t^{\alpha} + w_2 \cdot \text{Liquidity}_t^{\beta}
	•	\alpha > 1이면 정보 기반 거래자가 많아질수록 변동성이 급격히 상승.
	•	\beta < 1이면 유동성이 높아질수록 변동성이 완화됨.
	3.	시장 충격 함수 기반 스캘핑 전략
	•	특정 수준 이상 가격이 상승(또는 하락)한 후, 정보 기반 트레이더들이 포지션을 정리할 때 가격이 평균으로 회귀하는 패턴을 활용.
	•	비선형 가격 반응 모델:
\Delta P_t = \lambda Q_t^\gamma
	•	0 < \gamma < 1이면 가격이 완만하게 조정됨 → 스캘핑 전략 가능

🔍 추가 고려 사항
	•	주문장(Level II) 데이터 활용 필수 → 비대칭 정보 감지는 틱 데이터 기반의 정밀 분석 필요
	•	시장 조성자와의 경쟁 고려 → 유동성 공급자가 빠르게 대응하는 시장에서는 전략 조정 필요
	•	옵션 시장과의 연계 분석 → 옵션 미결제약정(OI) 및 변동성 스마일 구조를 활용하면 신호의 신뢰도 증가

📢 결론
비대칭 정보 이론을 활용한 비선형 퀀트 전략은 고빈도 트레이딩(HFT), 변동성 예측, 시장 미시구조 분석 등 다양한 분야에 적용 가능하며, 특히 주문 흐름 기반의 시장 충격 모델과 결합하면 강력한 알파를 창출할 수 있다. 🚀
