📌 시장 미시 구조 이론: 정보 엔트로피 기반 유동성 피드백 이론 (Information-Entropy Liquidity Feedback Theory)

🏛 이론 개요

정보 엔트로피 기반 유동성 피드백 이론은 호가 과정에서 정보 엔트로피가 높아질수록 주문 분산이 커지고, 이는 유동성 회수와 공급의 비대칭성을 유발한다는 가설입니다.

- 참가자들은 정보 불확실성이 커질수록 방어적 주문 전략을 취하며,
- 이때 유동성 공급과 회수 간의 비선형 피드백이 발생하여 가격 반응이 증폭됩니다.

⸻

📈 비선형 퀀트 알파 생성 방법
1. 실시간 정보 엔트로피 추정
   H_t = -\sum_i p_i \log p_i
2. 엔트로피 기반 유동성 피드백 계수
   L_t = \frac{H_t}{\sigma_t + \epsilon}
3. 임계값 \theta 를 초과하는 경우 유동성 공급 축소에 베팅하는 포지션을 구성합니다.
